Applications mobiles

Applications essentielles en Chine

Un annuaire complet des applications dont vous avez besoin en Chine : WeChat, Alipay, DiDi, Baidu Maps, Meituan, 12306, VPN et plus encore. Chaque application inclut des liens de téléchargement App Store / Google Play et des versions web H5 utilisables sans installation.

Avant de partir : Installez ces applications en WiFi avant le départ. Certaines boutiques d’applications sont lentes ou bloquées depuis la Chine. Téléchargez chez vous les packs de traduction hors ligne et les applications VPN.

Paiement et portefeuille

Alipay

Le portefeuille mobile le plus accepté en Chine. Associez une carte Visa ou Mastercard étrangère et payez par QR code dans les magasins, restaurants, taxis et stands de rue.

App Store Google Play

Astuce : La configuration prend 10 à 15 min. Le passeport est nécessaire pour la vérification.

Lire le guide de configuration →

WeChat

La super app chinoise. Utilisez-la pour les messages, les appels, Moments (fil social) et WeChat Pay. La plupart des Chinois l’utilisent tous les jours pour tout.

App Store Google Play Ouvrir la version web

Astuce : WeChat Pay nécessite une carte associée. WeChat fonctionne pour la messagerie sans configuration particulière.

Lire le guide de configuration →

Transport et VTC

DiDi

L’équivalent chinois d’Uber. Réservez des taxis et des voitures privées. Interface en anglais disponible. Paiement possible avec Alipay ou WeChat Pay.

App Store Google Play Ouvrir la version web

Astuce : La version H5 fonctionne dans le navigateur ou le mini-programme WeChat. Aucune application nécessaire pour des trajets occasionnels.

Lire le guide de configuration →

Mini-programme transport Alipay

Dans Alipay, utilisez le mini-programme transport pour scanner les QR codes du métro et des bus dans la plupart des grandes villes.

Astuce : Ouvrez Alipay → icône « Transport ». Aucune installation séparée nécessaire.

Lire le guide de configuration →

12306

Application officielle de China Railway pour réserver des billets de train. Interface uniquement en chinois, avec vérification requise. Alternative : utiliser Trip.com.

App Store Google Play Ouvrir la version web

Astuce : Compliqué pour les étrangers. Utilisez Trip.com (anglais, moins de contraintes de vérification).

Lire le guide de configuration →

Trip.com (Ctrip)

Application adaptée aux anglophones pour réserver trains, vols et hôtels. Accepte directement les cartes étrangères.

App Store Google Play Ouvrir la version web

Astuce : Meilleure alternative à 12306 pour les billets de train. Frais de service plus élevés, mais beaucoup plus simple.

Navigation et cartes

Baidu Maps

La carte la plus détaillée pour la Chine. Navigation étape par étape, itinéraires en transports publics, recherche de restaurants. Uniquement en chinois, mais utilisable avec les gestes.

App Store Google Play Ouvrir la version web

Astuce : Google Maps est bloqué en Chine. Baidu ou Amap sont nécessaires pour la navigation.

Amap (Gaode)

Alternative à Baidu Maps. Beaucoup de locaux préfèrent Amap. Meilleur pour la navigation en taxi, le trafic en direct et les avis de restaurants.

App Store Google Play Ouvrir la version web

Astuce : Baidu et Amap fonctionnent tous les deux. Choisissez-en un. Amap est souvent plus précis pour les VTC.

Google Maps

Fonctionne partiellement en Chine pour consulter des cartes, mais la navigation et le trafic en temps réel sont peu fiables. Utile pour préparer le voyage.

Ouvrir la version web

Astuce : Ne fonctionnera pas pour la navigation étape par étape en Chine sans VPN.

Lire le guide de configuration →

Repas et restaurants

Meituan

Livraison de repas, réservation de restaurants, billets de cinéma, réservation d’hôtels. Uniquement en chinois. À utiliser dans le mini-programme Alipay pour obtenir de l’aide en anglais.

App Store Google Play Ouvrir la version web

Astuce : La livraison H5 fonctionne dans le navigateur. Payez avec Alipay à la livraison.

Lire le guide de configuration →

Ele.me

Application de livraison de repas (propriété d’Alibaba). Similaire à Meituan. Permet de commander avec Alipay.

App Store Google Play Ouvrir la version web

Astuce : Meituan et Ele.me fonctionnent tous les deux. Essayez les deux pour une meilleure couverture des restaurants.

Lire le guide de configuration →

Dianping

Le Yelp chinois. Avis, notes et photos de restaurants. Aide à trouver de bonnes adresses locales. Uniquement en chinois.

App Store Google Play Ouvrir la version web

Astuce : Utilisez la recherche par photo : voyez à quoi ressemblent les plats avant de commander.

Lire le guide de configuration →

Communication et VPN

VPN (choisissez-en un avant le voyage)

Installez un VPN AVANT de partir. ExpressVPN, NordVPN et Astrill fonctionnent en Chine. Les VPN gratuits sont généralement bloqués.

Astuce : ESSENTIEL : installez-le et testez-le avant l’arrivée. Activez-le dès l’atterrissage.

Lire le guide de configuration →

Google Translate

Traduction par caméra pour menus, panneaux et documents. Téléchargez le pack hors ligne chinois simplifié.

App Store Google Play Ouvrir la version web

Astuce : Meilleur outil de traduction. Téléchargez le pack hors ligne avant le voyage.

Lire le guide de configuration →

Réservations de voyage

Trip.com (Hotels)

Idéal pour réserver des hôtels avec des cartes étrangères. Large couverture. Interface en anglais.

Ouvrir la version web

Astuce : Fonctionne aussi pour les vols et les trains. Solution tout-en-un.

Fliggy (Alitrip)

Plateforme de réservation de voyage d’Alibaba. Intégrée à l’application Alipay. Utile pour les vols intérieurs.

Ouvrir la version web

Astuce : Paiement possible avec Alipay. Aucun téléchargement séparé nécessaire.

Applications web H5 — Aucun téléchargement nécessaire

De nombreux services chinois proposent des versions web (H5) qui fonctionnent directement dans le navigateur de votre téléphone ou dans les mini-programmes WeChat/Alipay. Elles sont très pratiques pour les courts séjours si vous ne voulez pas encombrer votre téléphone avec plus de 10 applications. Les principales :

DiDi — m.didiglobal.com (anglais disponible)
Baidu Maps — map.baidu.com (chinois uniquement)
WeChat Web — web.wechat.com (messagerie uniquement)
Meituan — waimai.meituan.com (livraison de repas)
Dianping — dianping.com (recherche de restaurants)
Mini-programmes Alipay — tout dans Alipay (aucune installation supplémentaire)

Les versions web fonctionnent sans VPN dans la plupart des cas. Pour les téléchargements d’applications, installez-les avant l’arrivée depuis votre WiFi à domicile.

Les liens vers les boutiques d’applications ouvrent les pages officielles. Les prix et la disponibilité peuvent varier selon les restrictions régionales.
