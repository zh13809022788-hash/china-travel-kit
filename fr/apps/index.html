Applications mobiles

Applications essentielles pour la Chine

Un répertoire complet des applications dont vous avez besoin en Chine — WeChat, Alipay, DiDi, Baidu Maps, Meituan, 12306, VPN et bien plus. Chaque application comprend des liens de téléchargement App Store / Google Play et des versions web H5 qui fonctionnent sans installation.

Avant de partir : installez ces applications en WiFi avant le départ. Certains stores d’applications sont lents ou bloqués depuis la Chine. Téléchargez chez vous les packs de traduction hors ligne et les applications VPN.

Paiement et portefeuille

Alipay

Le portefeuille mobile le plus accepté en Chine. Associez une Visa ou Mastercard étrangère, puis payez par code QR dans les magasins, restaurants, taxis et chez les vendeurs de rue.

App Store Google Play

Conseil : La configuration prend 10 à 15 min. Passeport requis pour la vérification.

Lire le guide de configuration →

WeChat

La super app de la Chine. Utilisez-la pour la messagerie, les appels, Moments (fil social) et WeChat Pay. La plupart des Chinois l’utilisent chaque jour pour tout.

App Store Google Play Ouvrir la version web

Conseil : WeChat Pay nécessite une carte associée. WeChat lui-même fonctionne pour la messagerie sans configuration.

Lire le guide de configuration →

Transport et VTC

DiDi

Le Uber de la Chine. Réservez des taxis et des voitures privées. Interface en anglais. Paiement possible avec Alipay ou WeChat Pay.

App Store Google Play Ouvrir la version web

Conseil : La version H5 fonctionne dans le navigateur ou via un mini-programme WeChat. Aucune application requise pour des trajets occasionnels.

Lire le guide de configuration →

Alipay Transport Mini-Program

Dans Alipay, utilisez le mini-programme de transport pour scanner les codes QR du métro et des bus urbains dans la plupart des grandes villes.

Conseil : Ouvrez Alipay → icône "Transport". Aucune installation séparée n’est nécessaire.

Lire le guide de configuration →

12306

Application officielle de China Railway pour réserver des billets de train. Interface uniquement en chinois, vérification requise. Alternative : utilisez Trip.com.

App Store Google Play Ouvrir la version web

Conseil : Compliquée pour les étrangers. Utilisez Trip.com (en anglais, sans tracas de vérification).

Lire le guide de configuration →

Trip.com (Ctrip)

Application conviviale en anglais pour réserver trains, vols et hôtels. Accepte directement les cartes étrangères.

App Store Google Play Ouvrir la version web

Conseil : Meilleure alternative à 12306 pour les billets de train. Frais de service plus élevés, mais bien plus simple.

Navigation et cartes

Baidu Maps

La carte de la Chine la plus détaillée. Navigation pas à pas, itinéraires en transports publics, recherche de restaurants. Uniquement en chinois, mais utilisable avec les gestes.

App Store Google Play Ouvrir la version web

Conseil : Google Maps est bloqué en Chine. Il faut Baidu ou Amap pour la navigation.

Amap (Gaode)

Alternative à Baidu Maps. Beaucoup d’habitants préfèrent Amap. Meilleure pour la navigation en taxi, le trafic en direct et les avis sur les restaurants.

App Store Google Play Ouvrir la version web

Conseil : Baidu et Amap fonctionnent tous les deux. Choisissez-en un. Amap tend à être plus précis pour héler un taxi.

Google Maps

Fonctionne partiellement en Chine pour la consultation, mais la navigation et le trafic en temps réel ne sont pas fiables. Utile pour préparer le voyage.

Ouvrir la version web

Conseil : Ne fonctionnera pas pour la navigation pas à pas en Chine sans VPN.

Lire le guide de configuration →

Nourriture et restauration

Meituan

Livraison de repas, réservation de restaurants, billets de cinéma, réservation d’hôtels. Uniquement en chinois. À utiliser via le mini-programme Alipay pour une aide en anglais.

App Store Google Play Ouvrir la version web

Conseil : La livraison de repas H5 fonctionne dans le navigateur. Payez avec Alipay à la livraison.

Lire le guide de configuration →

Ele.me

Application de livraison de repas (appartenant à Alibaba). Similaire à Meituan. Vous pouvez commander à manger avec Alipay.

App Store Google Play Ouvrir la version web

Conseil : Meituan et Ele.me fonctionnent tous les deux. Essayez les deux pour la meilleure couverture de restaurants.

Lire le guide de configuration →

Dianping

Le Yelp de la Chine. Avis, notes et photos de restaurants. Aide à trouver de bonnes adresses locales. Uniquement en chinois.

App Store Google Play Ouvrir la version web

Conseil : Utilisez la fonction de recherche par photo — voyez à quoi ressemblent les plats avant de commander.

Lire le guide de configuration →

Communication et VPN

VPN (choisissez-en un avant le voyage)

Installez un VPN AVANT de partir. ExpressVPN, NordVPN et Astrill fonctionnent en Chine. Les VPN gratuits sont pour la plupart bloqués.

Conseil : ESSENTIEL : installez-le et testez-le avant votre arrivée. Activez-le quand vous atterrissez.

Lire le guide de configuration →

App Store Google Play Ouvrir la version web

Conseil : Le meilleur outil de traduction. Téléchargez le pack hors ligne avant le voyage.

Réservation de voyage

Trip.com (Hotels)

Le meilleur pour réserver des hôtels avec des cartes étrangères. Large couverture. Interface en anglais.

Ouvrir la version web

Conseil : Fonctionne aussi pour les vols et les trains. Solution tout-en-un.

Fliggy (Alitrip)

La plateforme de réservation de voyages d’Alibaba. À l’intérieur de l’application Alipay. Bien pour les vols intérieurs.

Ouvrir la version web

Conseil : Paiement possible avec Alipay. Aucun téléchargement séparé n’est nécessaire.

Applications web H5 — aucun téléchargement nécessaire

De nombreux services en Chine proposent des versions web (H5) qui fonctionnent directement dans le navigateur de votre téléphone ou dans les mini-programmes WeChat/Alipay. Elles sont idéales pour les voyageurs de courte durée qui ne veulent pas encombrer leur téléphone avec plus de 10 applications. Les principales :

DiDi — www.didiglobal.com (anglais disponible)
Baidu Maps — map.baidu.com (chinois uniquement)
WeChat Web — web.wechat.com (messagerie uniquement)
Meituan — waimai.meituan.com (livraison de repas)
Dianping — dianping.com (recherche de restaurants)
Alipay Mini-Programs — Tout dans Alipay (aucune installation supplémentaire)

Les versions web fonctionnent sans VPN dans la plupart des cas. Pour les téléchargements d’applications, installez-les avant votre arrivée via le WiFi de chez vous.

Les liens des stores d’applications ouvrent les pages officielles. Les prix et la disponibilité sont soumis aux restrictions régionales.
