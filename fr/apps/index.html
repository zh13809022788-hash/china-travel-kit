Applications mobiles

Applications indispensables en Chine

Un guide complet des applications utiles en Chine : WeChat, Alipay, DiDi, Baidu Maps, Meituan, 12306, VPN et bien d’autres. Chaque application comprend des liens App Store et Google Play, ainsi qu’une version web H5 utilisable sans installation.

Avant de partir : installez ces applications en Wi-Fi avant le départ. Depuis la Chine, certaines boutiques d’applications sont lentes ou bloquées. Téléchargez aussi à l’avance les packs de traduction hors ligne et vos applications VPN.

Paiement et portefeuille

Alipay

Le portefeuille mobile le plus largement accepté en Chine. Associez une carte Visa ou Mastercard étrangère et payez par QR code dans les magasins, restaurants, taxis et échoppes de rue.

App Store Google Play

Astuce : La configuration prend environ 10 à 15 minutes. Un passeport est nécessaire pour la vérification.

Lire le guide d’installation →

WeChat

La super-application chinoise par excellence. Utilisez-la pour la messagerie, les appels, Moments, le fil social, et WeChat Pay. La plupart des Chinois s’en servent tous les jours pour presque tout.

App Store Google Play Ouvrir la version web

Astuce : WeChat Pay nécessite une carte associée. La messagerie WeChat fonctionne, elle, sans configuration particulière.

Lire le guide d’installation →

Transport et VTC

DiDi

L’équivalent chinois d’Uber. Réservez des taxis et des voitures avec chauffeur. Une interface en anglais est disponible. Le paiement peut se faire avec Alipay ou WeChat Pay.

App Store Google Play Ouvrir la version web

Astuce : La version H5 fonctionne dans le navigateur ou via le mini-programme WeChat. Pour un usage occasionnel, aucune application n’est indispensable.

Lire le guide d’installation →

Mini-programme transport Alipay

Dans Alipay, le mini-programme transport permet de scanner les QR codes du métro et du bus dans la plupart des grandes villes.

Astuce : Ouvrez Alipay puis l’icône « Transport ». Aucune installation séparée n’est nécessaire.

Lire le guide d’installation →

12306

L’application officielle de China Railway pour réserver des billets de train. L’interface est uniquement en chinois et une vérification est requise. L’alternative la plus simple reste Trip.com.

App Store Google Play Ouvrir la version web

Astuce : Peu pratique pour les voyageurs étrangers. Mieux vaut utiliser Trip.com, en anglais et avec moins de contraintes de vérification.

Lire le guide d’installation →

Trip.com (Ctrip)

Une application pratique pour réserver trains, vols et hôtels avec une interface adaptée aux voyageurs internationaux. Les cartes étrangères sont acceptées directement.

App Store Google Play Ouvrir la version web

Astuce : La meilleure alternative à 12306 pour les billets de train. Les frais de service sont plus élevés, mais l’expérience est nettement plus simple.

Navigation et cartes

Baidu Maps

La carte la plus détaillée pour la Chine. Navigation pas à pas, itinéraires en transports publics et recherche de restaurants. L’interface est en chinois uniquement, mais reste utilisable avec les gestes.

App Store Google Play Ouvrir la version web

Astuce : Google Maps est bloqué en Chine. Pour se déplacer correctement, Baidu Maps ou Amap sont indispensables.

Amap (Gaode)

Une alternative à Baidu Maps. Beaucoup d’habitants préfèrent Amap. L’application est souvent plus pratique pour les trajets en taxi, le trafic en direct et les avis sur les restaurants.

App Store Google Play Ouvrir la version web

Astuce : Baidu Maps et Amap fonctionnent tous les deux. Choisissez-en une. Amap est souvent plus précise pour les VTC.

Google Maps

Fonctionne partiellement en Chine pour consulter des cartes, mais la navigation et le trafic en temps réel restent peu fiables. Utile surtout pour préparer le voyage avant le départ.

Ouvrir la version web

Astuce : La navigation pas à pas ne fonctionnera pas correctement en Chine sans VPN.

Lire le guide d’installation →

Repas et restaurants

Meituan

Livraison de repas, réservation de restaurants, billets de cinéma et hôtels. L’application est uniquement en chinois. Le plus simple est souvent de passer par le mini-programme Alipay pour obtenir un peu d’aide en anglais.

App Store Google Play Ouvrir la version web

Astuce : La version H5 de livraison fonctionne dans le navigateur. Vous pouvez payer avec Alipay à la livraison.

Lire le guide d’installation →

Ele.me

Application de livraison de repas appartenant à Alibaba. Elle ressemble beaucoup à Meituan et permet de commander avec Alipay.

App Store Google Play Ouvrir la version web

Astuce : Meituan et Ele.me se complètent bien. Tester les deux permet d’accéder à davantage de restaurants.

Lire le guide d’installation →

Dianping

Le Yelp chinois. Avis, notes et photos de restaurants pour repérer de bonnes adresses locales. Interface uniquement en chinois.

App Store Google Play Ouvrir la version web

Astuce : Servez-vous des photos pour voir à quoi ressemblent les plats avant de commander.

Lire le guide d’installation →

Communication et VPN

VPN (à choisir avant le départ)

Installez un VPN avant de partir. ExpressVPN, NordVPN et Astrill fonctionnent en Chine. Les VPN gratuits sont le plus souvent bloqués.

Astuce : Indispensable : installez-le et testez-le avant d’arriver. Activez-le dès l’atterrissage.

Lire le guide d’installation →

Google Translate

Traduction par caméra pour les menus, panneaux et documents. Téléchargez le pack hors ligne en chinois simplifié.

App Store Google Play Ouvrir la version web

Astuce : L’un des meilleurs outils de traduction. Téléchargez le pack hors ligne avant le voyage.

Lire le guide d’installation →

Réservations de voyage

Trip.com (Hôtels)

Très pratique pour réserver des hôtels avec des cartes étrangères. Couverture large et interface en anglais.

Ouvrir la version web

Astuce : Fonctionne aussi pour les vols et les trains. Une solution tout-en-un.

Fliggy (Alitrip)

Plateforme de réservation de voyages d’Alibaba, intégrée à l’application Alipay. Utile notamment pour les vols intérieurs.

Ouvrir la version web

Astuce : Le paiement peut se faire avec Alipay. Aucun téléchargement séparé n’est nécessaire.

Applications web H5 : aucun téléchargement nécessaire

De nombreux services chinois proposent des versions web H5 qui fonctionnent directement dans le navigateur de votre téléphone ou dans les mini-programmes WeChat et Alipay. Elles sont particulièrement pratiques pour les courts séjours si vous ne voulez pas surcharger votre téléphone avec plus de dix applications. Voici les principales :

DiDi — www.didiglobal.com (anglais disponible)
Baidu Maps — map.baidu.com (en chinois uniquement)
WeChat Web — web.wechat.com (messagerie uniquement)
Meituan — waimai.meituan.com (livraison de repas)
Dianping — dianping.com (recherche de restaurants)
Mini-programmes Alipay — tout est intégré dans Alipay, sans installation supplémentaire

Les versions web fonctionnent le plus souvent sans VPN. En revanche, pour les téléchargements d’applications, installez tout avant l’arrivée depuis votre Wi-Fi domestique.

Les liens vers les boutiques d’applications renvoient vers les pages officielles. Les prix et la disponibilité peuvent varier selon les restrictions régionales.
